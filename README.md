# 의약품 원료 가격 요약 시스템

원료 의약품의 구매 단가 변동을 추적하고, 월별 재무 영향을 분석하는 Streamlit 기반 웹 애플리케이션입니다.

## 개요

이 애플리케이션은 제약 회사의 원료 구매 담당자가 PO(Purchase Order) 기반으로 원료별 단가 변동을 손쉽게 파악하고, USD 기준으로 재무 영향을 정량화할 수 있도록 설계되었습니다.

---

## 주요 기능

| 기능 | 설명 |
|---|---|
| 거래 데이터 직접 입력 | 사이드바 폼을 통해 PO 단위 구매 내역 저장 |
| Excel 일괄 가져오기 | 기존 양식의 Excel 파일을 업로드하여 대량 데이터 등록 |
| 품목별 가격 등락 분석 | 직전 거래 대비 단가 변동률 및 상태(인상/인하/유지) 표시 |
| 월별 재무 영향 리포트 | 선택한 월의 가격 변화가 구매 비용에 미친 영향을 USD로 환산 |
| 가격 추세 시각화 | 품목별 단가의 시계열 변동 차트 제공 |
| 엑셀 보고서 다운로드 | 분석 결과를 xlsx 파일로 내보내기 |

| 단가 | float | 단위당 가격 | 12.50 |
| 총금액 | float | 구매수량 × 단가 | 6250.00 |

## 설치 및 실행

**1. 저장소 클론**

```bash
git clone <repository-url>
cd <project-folder>
```

**2. 의존성 설치**

```bash
pip install streamlit pandas altair xlsxwriter openpyxl
```

**3. 앱 실행**

```bash
streamlit run app.py
```

브라우저에서 `http://localhost:8501` 접속

## Excel 가져오기 양식

업로드할 Excel 파일은 아래 열 이름을 포함해야 합니다.

| Excel 열 이름 (필수) | 앱 내부 컬럼 |
|---|---|
| Tanggal PO | 날짜 |
| Nomor PO | PO_번호 |
| Kode Barang | 품목코드 |
| Nama Barang | 품목명 |
| Jumlah | 구매수량 |
| Satuan | 단위 (선택) |
| Mata Uang | 화폐 (선택, 기본값: USD) |
| Harga Satuan | 단가 |

> `단위`와 `화폐` 열이 없을 경우 각각 `-`, `USD`로 자동 설정됩니다.
> 중복 데이터는 `PO_번호 + 품목코드` 조합을 기준으로 최신 데이터로 덮어씁니다.
